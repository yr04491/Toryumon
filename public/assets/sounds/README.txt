このフォルダに音声ファイルを配置してください。
ファイル名は soundManager.ts の SOUNDS で参照しているものに合わせます。

必須ファイル：
  bgm_title.mp3   … タイトル画面のBGM（ループ）
  bgm_game.mp3    … ゲーム画面のBGM（ループ）
  se_correct.mp3  … 正しく指摘して校長が飛んでくるとき
  se_wrong.mp3    … 誤報告して先生に笑われるとき
  se_yamada.mp3   … 山田くんに先を越されたとき
  se_clear.mp3    … ステージクリア（リザルトへ進むとき）

※ ファイルが無くてもアプリは動作します（再生が無視されるだけ）。
※ 音量・ミュートはタイトル画面の「設定」ボタンから変更でき、localStorageに保存されます。
